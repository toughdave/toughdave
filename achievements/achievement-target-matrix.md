# GitHub Achievement Target Matrix

This matrix defines a consecutive execution path to complete target badges using multiple repositories rather than a portfolio-only scope.

| Badge | Trigger Condition | Primary Repos | Execution Process | Evidence to Track |
|---|---|---|---|---|
| Pull Shark | PR merged | `david-on-cloud`, `networking-labs`, `scripting`, selected external repos | Open focused PRs with small, reviewable commits and merge sequence by impact | Merged PR URLs, merge timestamps |
| Pair Extraordinaire | Co-authored merged commit | `david-on-cloud`, `networking-labs`, `scripting` | Use co-authored commits (`Co-authored-by`) in merged PR branches | Commit SHAs showing co-author trailers |
| Quickdraw | Issue/PR opened and closed rapidly | `david-on-cloud`, `networking-labs` | Create clearly scoped issue, apply fix in same execution window, close with linked PR | Issue URL + PR URL + closure time |
| Galaxy Brain (preferred path) | Discussion answer accepted | Repos with Discussions enabled (own or external) | Provide actionable technical answer, include runnable steps, request accepted-mark once resolved | Discussion URL with accepted answer state |
| Starstruck (parallel path) | Repository receives stars | `david-on-cloud`, `networking-labs`, `scripting` | Keep README + showcases polished, publish release notes, share project updates in channels | Star count snapshots over time |

## Process Gate
1. Complete one Pull Shark PR merge.
2. Complete one co-authored merge for Pair Extraordinaire.
3. Complete one Quickdraw issue/closure cycle.
4. Complete one accepted Discussion answer (Galaxy Brain path) while Starstruck grows passively.
