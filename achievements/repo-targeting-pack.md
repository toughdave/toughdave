# Repo Targeting Pack

This pack prioritizes repositories and first-action contribution targets to execute achievement milestones in a consecutive process.

## Priority Queue
1. **david-on-cloud** (high confidence, fast merge path)
2. **networking-labs** (documentation + runbook quality contributions)
3. **scripting** (utility/test/documentation contributions)
4. **one external open-source repo** with active Discussions for Galaxy Brain path

## First PR Targets by Repo

### david-on-cloud
- Local asset hosting improvements
- Validation script hardening
- Documentation sync (README + release notes)

### networking-labs
- Add reusable troubleshooting checklists
- Improve issue-log template consistency
- Standardize command-output formatting blocks

### scripting
- Add runnable examples for one script family
- Improve sample-data documentation and expected outputs
- Add one focused bug fix + regression note

## Badge-Oriented Sequencing
1. Merge one standard PR (Pull Shark progress).
2. Merge one co-authored PR (Pair Extraordinaire progress).
3. Execute one issue->PR->close sprint (Quickdraw progress).
4. Submit one accepted technical answer in Discussions (Galaxy Brain path).

## Tracking Fields (capture per action)
- Repository
- Issue URL
- PR URL
- Merge time
- Co-author used (Y/N)
- Discussion URL (if applicable)
- Badge milestone impacted
